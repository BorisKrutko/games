# Конструкторный замес 

|число игроков|14+|
|необходимо|конструктор, веревки, свисток, флажки для 2х команд|
|время игры|3-5 мин. раунд|

## Цель игры: собрать как можно больше конструктора

Есть 3 зоны: нейтральный коридор по центру, с конструктором, и зоны 2х команд по бокам. Команды выбирают 3х лучников, остальные пешки. Никто не может зайти на территорию противника (в нейтральный коридор может зайти каждый). Задача пешки собирать конструктор из центра, мяча пешка касаться не может. За один раз, пешка может взять лишь 1 конструктор, зайдя 2-мя ногами в нейтральный коридор, собранный конструктор, пешка собирает в ЛИЧНУЮ башню у стенки. Задача лучников: выбивать пешки, выбить можно везде (нейтральный коридор / зона противника), т.е. спрятаться нигде нельзя, могут выбить пешку всегда. В сл. если пешку выбили, пешка сдает свою ЛИЧНУЮ башню в ведро противника, расположенное на боковой линии игрового поля (уже гарантированный очки). После того как прозвучал свисток, считается кол-во собранного конструктора в ведре + ЛИЧНЫЕ башни. Когда пешку выбивают, она становится лучником [ЛУЧНИК = платок на шее, ПЕШКА = ПЛАТОК НА РУКЕ]

Игра заканчивается, если выбила всех пеше одной из команд, либо закончился конструктор в коридоре.
